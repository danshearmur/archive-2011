Last weekend I accomplished something, that I have been meaning to do for well over a year, install Mac OS X on my PC. I had tried a fewdifferent methods that never really got very far; thank god for tonymacx86 and his post:

iBoot + MultiBeast: Install Mac OS X on any Intel-based PC.

Turns out it was easier than installing Windows 7. The two drivers missing under OS X were audio and wireless; for which I used VoodooHDA and the Ralink drivers respectively. All in all it’s been a very pleasant experience.

I’ve had quite a lot of differing advice about what’s good and what’s not; so I’m gonna wait before I start shouting about what I like on the platform. The one thing I really need to draw attention to is how different the keyboard shortcuts are!

Some choice shortcuts:

  • Command+Arrow Down to open a file or folder in finder
  • Command+Backspace to delete a file in finder
  • Option+Arrows to navigate words at once in text (think control and arrow key on Windows)
  • Command+Shift+Backspace to clear trash can (I miss my shift+del; proper delete!)
