BindF5

Use F5 to refresh your browser on Mac OS X

This script allows you to refresh your browser with the F5 key, or alternatively reload every element on the page with Ctrl+F5.

Tested on:

  • Chrome 5
  • Firefox 3.6.7 with Greasemonkey 0.8
  • Safari 5 with GreaseKit 1.7

It should hopefully work on websites that use frames as well.

People from a windows background may find this useful!

Install

Chrome

  1. Go to userscripts.org and click "Install"
  2. Click Continue from the status bar
  3. Click Install from the dialog box

Firefox

(Firefox supports F5 normally, but not Ctrl+F5)

  1. Install Greasemonkey
  2. Go to userscripts.org and click "Install"
  3. Click Install again from the dialog box

Safari

  1. Download and Install SIMBL
  2. Download GreaseKit
  3. Copy GreasKit.bundle to ~/Library/Application Support/SIMBL/Plugins
  4. Relaunch Safari
  5. Go to userscripts.org and click "Install"
  6. Click Install again from the dialog box
